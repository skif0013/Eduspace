🎓 EduspaceFron

Frontend-часть проекта Eduspace — современного образовательного веб-приложения, созданного на React + Vite + TailwindCSS.

🚀 Установка и запуск проекта

Следуй шагам ниже, чтобы запустить проект локально 👇

1️⃣ Клонируй репозиторий
git clone https://github.com/твоя-ссылка-на-репозиторий.git

2️⃣ Перейди в папку проекта
cd EduspaceFron

3️⃣ Установи все зависимости
npm install


или

npm i

4️⃣ Запусти проект в режиме разработки
npm run dev


После запуска терминал покажет адрес (обычно):

http://localhost:5173/


Открой его в браузере 🌐

🧩 Технологии

⚛️ React — UI-библиотека

⚡ Vite — быстрый сборщик

🎨 TailwindCSS — стилизация

🧭 React Router DOM — маршрутизация
